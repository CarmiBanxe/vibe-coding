# Vibe Coding Project

Рабочий репозиторий для совместной разработки через Perplexity + GitHub.

## Как работает

1. **Perplexity** создаёт и коммитит код в этот репозиторий
2. **Ты** запускаешь `sync.bat` на рабочем столе — файлы скачиваются на Legion
3. Открываешь проект в VS Code и работаешь

## Структура

```
vibe-coding/
├── README.md          — этот файл
├── docs/              — документация, саммари, планы
├── src/               — исходный код
└── scripts/           — вспомогательные скрипты
```
