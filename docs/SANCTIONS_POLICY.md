Транзакции и клиенты из этих юрисдикций **отклоняются автоматически**.

## Правила применения

1. **HARD REJECT**: немедленный отказ, без анализа, без передачи субагентам
2. **EDD**: передать агенту `compliance`, обязательно HITL (Compliance Officer)
3. **Списки динамичны**: FATF обновляет трижды в год (февраль, июнь, октябрь)
4. Транзакции в USD через американскую инфраструктуру — применяются санкции OFAC даже для не-американских контрагентов
