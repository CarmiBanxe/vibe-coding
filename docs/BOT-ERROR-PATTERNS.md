# BOT-ERROR-PATTERNS.md — Banxe AI Bank
> Паттерны ошибок в коде, сгенерированном ботом
> Обновляется ctio при накоплении 3+ однотипных ошибок
> Бот читает этот файл через workspace и избегает повторения

---

## Правило использования

Когда верификация находит ошибку в коде от бота:
1. ctio помечает находку: "от бота"
2. Добавляет запись ниже после 3+ однотипных случаев
3. Бот читает при следующей сессии — не повторяет паттерн

---

## Паттерны ошибок

_Пока пусто — заполняется по мере работы_

```
### [YYYY-MM-DD] название-паттерна
- Инструмент: Semgrep / CodeRabbit / CodeQL
- Правило: rule-id
- Паттерн-ошибка: что именно бот делает неправильно
- Правильный подход: как надо
- Встречалось: N раз
```
