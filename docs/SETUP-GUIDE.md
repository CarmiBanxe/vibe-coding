# Пошаговая инструкция: первая настройка на Legion

## Что нужно сделать ОДИН РАЗ

### Шаг 1 — Скачай sync.bat на рабочий стол

1. Открой браузер на Legion
2. Перейди по ссылке: https://github.com/CarmiBanxe/vibe-coding
3. Нажми на файл `sync.bat`
4. Нажми кнопку **Download raw file** (иконка скачивания ↓ справа)
5. Сохрани файл на **Рабочий стол**

### Шаг 2 — Запусти sync.bat

1. Дважды кликни по `sync.bat` на рабочем столе
2. Откроется чёрное окно (командная строка)
3. Скрипт автоматически скачает проект в папку `C:\Users\ТВОЁ_ИМЯ\vibe-coding`
4. После скачивания автоматически откроется VS Code с проектом

**Если Windows спросит "Защита SmartScreen"** — нажми "Подробнее" → "Выполнить в любом случае"

### Шаг 3 — Готово!

Теперь каждый раз, когда я скажу "обнови проект" или "сделал изменения" — просто дважды кликни `sync.bat`.

## Если что-то пошло не так

### "git не распознан как команда"
Git не установлен или не добавлен в PATH. Открой PowerShell и набери:
```
winget install Git.Git
```
Перезагрузи компьютер и попробуй снова.

### "Ошибка аутентификации"
Git не знает твой GitHub аккаунт. Открой PowerShell и набери:
```
git config --global user.name "CarmiBanxe"
git config --global user.email "yachta315@gmail.com"
```
Затем запусти sync.bat — при первом клоне GitHub попросит авторизоваться в браузере.
