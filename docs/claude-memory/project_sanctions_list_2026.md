# Страны с запрещёнными/ограниченными финансовыми транзакциями (апрель 2026)

## Ключевые изменения 2025–2026
- **Россия** → EU AML High-Risk list с 29.01.2026 (двойное основание: OFAC/EU/UK санкции + AML HR)
- **Сирия** → OFAC comprehensive sanctions сняты с 01.07.2025 (EO 14312); ЕС и UK также сняли большинство ограничений. Остаются SDN-листинги Асада и окружения → статус: **HOLD + ручная проверка**, не BLOCK
- **Боливия, BVI** → добавлены в EU AML High-Risk + FATF greylist (2025)
- Удалены из FATF greylist (окт. 2025): Буркина-Фасо, Мозамбик, Нигерия, ЮАР

## Четыре регуляторных уровня (UK FCA EMI — все обязательны)
1. **OFAC** — через USD-корреспондентство и вторичные санкции
2. **UK OFSI / UK Sanctions List** — обязателен (с 28.01.2026 — UK Sanctions List)
3. **EU Sanctions** — при работе через EUR-расчёты
4. **FATF / EU AML High-Risk** — требует EDD, не полного запрета (FATF Grey List ≠ блокировка)

## 5 ключевых фактов для бота

1. Россия — двойное основание: санкции + EU AML HR (с января 2026)
2. Сирия — BLOCK → HOLD (с июля 2025), Banxe список не обновлён
3. FATF grey list ≠ запрет; это EDD, не блокировка
4. UK EMI применяют более широкие списки чем требуют регуляторы (давление payment networks)
5. Вторичные санкции OFAC применяются к любым non-US финтех через USD или US counterparties
